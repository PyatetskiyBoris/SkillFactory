Проект 2. Разведывательный анализ данных

Основные цели и задачи проекта:

Провести анализ данных с целью очистки и отбора данных для построения дальнейшей модели машинного обучения.

Краткая информация о данных:

Таблица содержит данные об условиях жизни учеников и их оценки госэкзамена по математике

Описание параметров в таблице содержится в файле exam.txt

Этапы работы над проектом:

1) Первичная обработка данных. Обработка выбросов и явно ошибочных значений

2) Корреляционный анализ количественных переменных, отбор столбцов для построения модели

3) Анализ номинативных переменных, отбор столбцов для построения модели
